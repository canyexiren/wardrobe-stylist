### 阶段6：穿搭生成 ✅
- [x] `wardrobe-app/js/outfit.js` — 完整穿搭引擎（~280行）：
  - `initStyleSelector()` — 风格选择器（3选1，保存默认到IndexedDB）
  - `loadDefaultStyle()` — 读取用户默认风格
  - `tempToWarmth(temp)` — 温度→保暖度映射（>30薄款，>20薄/适中，>10适中，>0厚款，≤0加厚）
  - `weatherNeedsOuterwear(code)` — 雨雪雾恶劣天气判断
  - `colorsHarmonize(c1, c2)` — 颜色协调打分（同色系+2，中性+1，混搭0）
  - `generateOutfit()` — 核心算法：保暖度+风格筛选→分类分组→随机挑→连衣裙替代→放宽降级
  - `renderOutfit(outfit)` — 渲染穿搭结果卡片（缩略图+名称+颜色+换一换按钮）
  - `swapItem(category)` — 换一换（从候选池排除当前，随机换）
  - 三种状态：空衣橱、无匹配（含保暖度建议）、正常穿搭
- [x] `wardrobe-app/index.html` — 风格选择器 + 生成按钮 + 结果区
- [x] `wardrobe-app/css/style.css` — ~160行穿搭样式（style-chip紫色高亮、生成按钮渐变、穿搭条目卡片）

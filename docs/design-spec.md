# UI 设计规范

> 版本: v1.0 | 日期: 2026-06-07

## 1. 设计理念

**关键词**: 简洁、柔和、呼吸感

整体采用白色背景 + 低饱和莫兰迪色系点缀，营造干净、温柔、不刺眼的视觉体验。

---

## 2. 色彩系统

### 2.1 主色板

| 用途 | 色值 | 色卡 | 说明 |
|------|------|------|------|
| 页面背景 | `#F8F7F4` | 暖白 | 全页背景，柔和护眼 |
| 卡片背景 | `#FFFFFF` | 纯白 | 卡片、弹窗背景 |
| 主文字 | `#3C3C3C` | 深灰 | 主要文字内容 |
| 辅助文字 | `#9B9B9B` | 中灰 | 描述、提示文字 |
| 分割线 | `#EEEDEA` | 浅灰 | 卡片边缘、分割线 |

### 2.2 点缀色（莫兰迪色系）

| 用途 | 色值 | 色卡 | 说明 |
|------|------|------|------|
| 主强调色 | `#A3B5A6` | 灰绿 | 主要按钮、高亮Tab |
| 次强调色 | `#B8A9C9` | 灰紫 | 标签、次要按钮 |
| 暖色点缀 | `#D4B5A1` | 灰粉 | 通勤风格标识 |
| 中性点缀 | `#B0B7BC` | 雾蓝 | 休闲风格标识 |
| 冷色点缀 | `#C4C0B7` | 灰棕 | 精致风格标识 |

### 2.3 功能色

| 用途 | 色值 | 说明 |
|------|------|------|
| 成功/确认 | `#7CAE7A` | 保存成功 |
| 警告 | `#E0B86C` | 删除警告 |
| 错误 | `#D49595` | 错误提示 |

## 3. 字体规范

| 层级 | 字号 | 字重 | 用途 |
|------|------|------|------|
| H1 页面标题 | 22px | 600 | 各Tab页面大标题 |
| H2 区域标题 | 18px | 500 | 天气温度数字 |
| Body 正文 | 15px | 400 | 衣物名称、按钮文字 |
| Caption 辅助 | 13px | 400 | 分类标签、提示文字 |
| Small 小字 | 11px | 400 | 时间戳、备注 |

**字体栈**: `-apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC", "Microsoft YaHei", sans-serif`

---

## 4. 间距系统

采用 4px 基准网格：

| 档位 | 值 | 用途 |
|------|-----|------|
| xs | 4px | 标签内边距 |
| sm | 8px | 卡片内边距、图标间距 |
| md | 16px | 页面内边距、卡片间距 |
| lg | 24px | 区块间距 |
| xl | 32px | 页面顶部/底部留白 |

---

## 5. 圆角规范

| 元素 | 圆角值 | 说明 |
|------|--------|------|
| 大卡片 | `12px` | 穿搭结果卡片、衣物卡片 |
| 小标签 | `6px` | 分类筛选标签、风格标签 |
| 按钮 | `24px` | 生成穿搭按钮（胶囊形） |
| 弹窗 | `16px` | 添加/编辑表单弹窗 |

## 6. 组件样式

### 6.1 底部导航栏

```
┌────────────┬────────────┬────────────┐
│    ✨      │    👗      │    👤      │
│  穿搭灵感  │  我的衣橱  │  个人中心  │
│  (active)  │            │            │
└────────────┴────────────┴────────────┘
```
- 高度: 64px (含安全区)
- 背景: `#FFFFFF`
- 上阴影: `0 -1px 8px rgba(0,0,0,0.06)`
- 选中态: 主强调色图标+文字
- 未选中: 辅助文字色

### 6.2 风格选择器

三个横向排列的标签按钮：
- 未选中: 白色背景 + 浅灰边框
- 选中: 主强调色背景 + 白色文字
- 宽度均分，间距 8px

### 6.3 衣物卡片

```
┌──────────────────┐
│                  │
│    [照片区域]     │  ← 固定高宽比 4:3
│                  │
├──────────────────┤
│  白色T恤          │  ← 15px 正文
│  通勤 · 休闲      │  ← 11px 辅助色标签
└──────────────────┘
```
- 圆角: 12px
- 阴影: `0 2px 8px rgba(0,0,0,0.06)`

### 6.4 按钮

**主按钮**（生成穿搭）:
- 背景: 主强调色
- 文字: 白色
- 圆角: 24px（胶囊形）
- 高度: 48px
- 宽度: 撑满容器

**浮动添加按钮**（衣橱页）:
- 圆形，直径 56px
- 右下角固定位置
- 背景: 主强调色
- 图标: "+" 白色

### 6.5 弹窗

```
┌──────────────────────┐
│  添加衣物        ✕   │  ← 标题栏
├──────────────────────┤
│                      │
│  [表单内容区域]       │
│                      │
├──────────────────────┤
│      [保存按钮]       │  ← 底部操作区
└──────────────────────┘
```
- 居中弹出
- 背景半透明遮罩
- 内容区可滚动
- 最大高度 80vh

---

## 7. 响应式策略

| 屏幕宽度 | 布局 |
|----------|------|
| < 480px (手机) | 全宽，卡片单列 |
| 480-768px (平板) | 最大宽度 480px 居中，卡片两列 |
| > 768px (桌面) | 最大宽度 600px 居中，卡片三列 |

- 所有页面限制最大宽度，桌面端居中显示
- 衣橱网格自适应列数
- 底部导航始终保持全宽

---

## 8. 动效规范

- Tab切换: 内容区淡入 `opacity 0.2s ease`
- 弹窗进出: 缩放+淡入 `transform 0.25s ease, opacity 0.25s ease`
- 按钮点击: 轻微缩放 `transform: scale(0.97)` + `transition: 0.1s`
- 卡片悬停: 轻微上浮 `transform: translateY(-2px)` + 阴影加深
